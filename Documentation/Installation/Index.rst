============
Installation
============

Target group: **Administrators**

.. important::

The extension needs to be installed as any other extension of TYPO3 CMS:

#. Switch to the module “Extension Manager”.

#. Get the extension

	#. **Use composer**: Use `composer require ashokatree/atvslider` OR `composer require ashokatree/atvslider 10.0.x-dev`

	#. **Get it from typo3.org:** You can always get current version from
      `http://typo3.org/extensions/repository/view/at_vslider/current/
      <http://typo3.org/extensions/repository/view/at_vslider/current/>`_ by
      downloading either the t3x or zip version. Upload
      the file afterwards in the Extension Manager.

Latest version from git
-----------------------
You can get the latest version from git by using the git command:

.. code-block:: bash

   git clone https://github.com/nirmalyamondal/at_vslider
